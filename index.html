FlameBot v2.0 – Autonomous Fire-Fighting Mobile Robot

FlameBot v2.0 - Hardware Chassis with 3-DOF Robotic Manipulator
Real-Time Web Dashboard - Fire & Human Detection utilizing YOLOv8
Autonomous SLAM Mapping & Path Planning Grid

FlameBot v2.0 is an advanced fire-fighting mobile robot designed to navigate indoor environments autonomously, locate fire sources, detect human presence, and suppress fires using an onboard arm manipulator and pump system.

Project Scope & Implementation:

  • Mechanical Design: Built a 4-wheeled mobile base featuring a custom 3-Degrees-of-Freedom manipulator arm and water hose integration.
  • SLAM & Navigation: Utilized Lidar sensors for real-time Simultaneous Localization and Mapping, generating high-fidelity occupancy grids for pathfinding.
  • Web Control Center: Coded a responsive monitoring dashboard featuring direct hardware controls, live camera feeds, and connection metrics.
  • AI Fire & Human Detection: Deployed a customized YOLO model to identify fires and people in real-time, sending alerts to local emergency response teams.
